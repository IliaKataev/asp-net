
Компоненты:

1. Карточка товара +
2. Шапка сайта +-
3. Корзина +-
4. Подвал +-

Страницы:
1. Главная (где все товары) +
2. Корзина +

Как это будет работать?
1. Пользователь должен видеть список товаров +
2. Добавить кнопку "добавить в корзину" на карточке товара +
3. Необходимо запомнить товары в корзине (локально) +
4. Открытие страницы со списком товаров при нажатии на кнопку "корзина" +

Вторая часть.

1. Поиск-компонент +
2. Передача произвольного набора атрибутов в карточку товара + 
3. Двусторонняя привязка для количества товара
4. Передача валюты через каскадный параметр
5. Жизненные циклы компонентов. Сообщение при загрузке товара
6. Внеднерие зависимостей. Внедрить сервис валюты
7. Оформление заказа, т.е. создать модель и привязать ее к форме
8. Оптимизация юзингов, т.е. убрать лишние. Добавить в imports
9. Заголовок страницы для корзины
